# Android Log Analyzer

## Quick Reference

| Component | Language | Purpose |
|-----------|----------|---------|
| Core | Rust | Log parsing & analysis |
| CLI | Go | Command line interface |
| API | Python | HTTP server |
| Web | JS | Browser UI |

## Common Commands

```bash
# Build
make build

# Run analysis
./bin/loganalyzer log.txt

# Start API
make run-api
```
